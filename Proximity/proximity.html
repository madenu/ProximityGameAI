Proximity

Proximity is a game originally created by Brian Cable. You can find an original Adobe Flash implementation of the game by querying an internet search engine. An HTML/JavaScript version of Proximity, written by Rod Pierce, can be found at mathisfun.com. Also, a basic Python version of the game can be found here.

This project modifies the HTML version to use standard AI techniques to make a smarter computer opponent.
